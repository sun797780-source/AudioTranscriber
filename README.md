# 🎧 在线听力转录工具 (Audio Transcriber)

> **专为学生党设计的听力神器！** 本工具支持一键下载、双击安装，支持通过 URL 链接（如 OneClass 听力网址）直接提取听力文本。

---

## ✨ 核心功能
- **一键安装环境**：自带 `install.bat`，自动安装 Python 3.10、ffmpeg 和所有依赖，无需手动配置。
- **高精度转录**：基于 OpenAI Whisper `base` 模型，支持多国语言。
- **极速下载**：支持多线程并发下载，大文件分分钟搞定。
- **无乱码保障**：针对 Windows 终端优化，完美显示中文。

---

## 🛠️ 第一步：环境准备（仅需一次）
**此项目需要运行在 Windows 10/11 上。**
1.  **Python**：若电脑内没有 Python，运行 `install.bat` 时会自动尝试帮你安装。
2.  **ffmpeg**：这是核心音频处理引擎。建议在运行 `install.bat` 时观察输出，如果自动安装失败，请按照脚本内的提示进行手动操作。

---

## 📖 第二步：实战教学 —— 如何获取音频 URL？
以 **OneClass (电脑端)** 为例，普通用户只需三步即可拿到音频地址：

1.  **打开听力页面**：在浏览器（推荐 Chrome 或 Edge）中打开你需要转录的听力页面。
2.  **呼出 F12 控制台**：
    - 在页面空白处点击【右键】->【检查】，或者直接按键盘上的 `F12` 键。
    - 在弹出的侧边栏/底部栏中，点击顶部的 **网络 (Network)** 选项卡。
3.  **筛选并获取链接**：
    - 在下方的搜索框或筛选器里，点击 **媒体 (Media)**。
    - **播放/刷新页面**：按一下听力的播放按钮。此时下面会出现一个列表，会有以 `.mp3`, `.wav`, `.m4a` 或带一长串字母的链接。
    - **复制地址**：在那个链接上【右键】->【复制】->【复制链接地址 (Copy Link Address)】。

**这就是我们要填入运行窗口的「音频 URL」。**

---

## 🚀 第三步：运行与转录

1.  **安装依赖**：双击运行 `install.bat`。它会自动创建虚拟环境并下载转录所需的各种库（使用清华镜像，速度极快）。
2.  **启动转录**：
    - 双击运行 `run.bat`。
    - 看到提示「请输入音频URL:」后，**鼠标右键**点击粘贴刚才复制的链接，回车。
    - 程序会自动下载并开始转录。
3.  **获取结果**：转录完成后，文字会实时显示在黑色窗口中。

---

## ⚠️ 注意事项
- **首次启动慢**：第一次运行转录时，程序会自动下载 140MB 的模型文件，请耐心等待 1-2 分钟。
- **显卡加速**：如果有 NVIDIA 显卡驱动，程序会自动调用显卡进行运算，速度更快。
